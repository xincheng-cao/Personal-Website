The idea of the Drawing Board is to detect, locate and track the fingertip through a video sequence accurately and efficiently. The main task is to detect userâ€™s hand gestures, so that Drawing Board can know whether it should draw or not. Also, the Drawing Board enables user to draw thick line or thin line. In order to distinguish the different purpose of users when they draw on Drawing Board, four hand gestures are predefined which represent four possible purpose of users, using thin paint brush to draw, using thick paint brush to draw, using thin paint brush but not drawing (just moving around or adjusting the hand position) and using thick paint brush but not drawing.
